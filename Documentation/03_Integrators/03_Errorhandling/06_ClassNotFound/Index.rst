Class not found
===============

Sometime you get the error prompt, class could not be found.

Please disable the extension with the missing class:

* :ref:`reset a package <integrators-errorhandling-resetpackage>` and 

* :ref:`clear the cache <integrators-errorhandling-cache>`

Enable the extension. Bug should be solved.
