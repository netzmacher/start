News (EXT:tx_news)
==================

Start TYPO3 Responsive can extend news images (EXT:tx_news) with
a responsive lightbox feature.

This feature is based on the responsive framework Foundation.


Setup
-----

You have to include the static Template:

* [Start [20.5.1] txnews images +lightbox (Foundation)]


Caution!
--------

If you include the template from above, the HTML of all news images (tx_news) 
of the entire webiste will changed.

If you have any unexpected behaviour, please remove the template.
