Modals
======

Newsletter
----------

If you are interested in a ready-to-use newsletter popup, please refer to

* :ref:`Administrators > Best Practise > Newsletter > Modal <administrators_bestpractise_newsletter_modal>`

* :ref:`Administrators > Best Practise > Newsletter > Popup <administrators_bestpractise_newsletter_popup>`


Content Element
---------------

Add to your root page a page "Modals".

* Set both Backend Layouts – this page and subpages – to [Start Simple 1: | Main | ]

* Move the page type from [Default] to [Folder].

Add to the folder "Modals" the folder "Contact" (in my sample the folder got the page id 8780).

Add to the content area of the folder "Contact" a content element:

* General

  * Type: [Text & Images]

  * Header: Contact

  * Text: Your name, phone number and e-mail-address.

* Images

  * Upload a portrait of you.

* Appearance

  * Image Adjustments: Width: 100

  * Image Alignment: [Beside Text, Left]

  * Number of columns: [1]


Link with the Icon Bar
----------------------

Link the content with an icon of the Icon Bar Start (see section "Icon Bar" on page 13 above).

Configure the Icon Bar Start with the Constant Editor.

* Category [START - FOUNDATION – ICONBAR START]

* 20 pid: page id of the modal folder contact (here: 8780)
